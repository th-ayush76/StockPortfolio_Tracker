# Stock Portfolio Tracker 📈

A Python-based application designed to help you track your investments, monitor portfolio performance, and manage your stock holdings efficiently.

## 🚀 Features
* **Portfolio Overview:** Keep track of all your purchased stocks in one place.
* **Performance Tracking:** Calculate your total investment, current value, and profit/loss.
* **Easy Management:** Simple command-line interface to add, remove, or view your stocks.

## 🛠️ How to Run
1. Ensure you have [Python](https://www.python.org/) installed on your system.
2. Clone this repository or download the source code.
3. Open your terminal or command prompt in the project folder.
4. Run the application using the following command:
```bash
   python StockPortfolioTracker.py
